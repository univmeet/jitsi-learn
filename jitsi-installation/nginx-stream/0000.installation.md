# libnginx-mod-stream

## 查看命令

```
dpkg -L libnginx-mod-stream | grep "nginx"
find / -name "*stream*" | grep "nginx" | grep -v "geoip2" | grep -v "fair"

ll /var/lib/dpkg/info/libnginx-mod-stream.*
sz /var/lib/dpkg/info/libnginx-mod-stream.list
```
