# jitsi-meet-web

## 查看命令

```
# 查看安装明细
dpkg -L jitsi-meet-web | grep "jitsi"
find / -name "*jitsi*" | grep "web" | grep -v "config"

ll /var/lib/dpkg/info/jitsi-meet-web.*
sz /var/lib/dpkg/info/jitsi-meet-web.list
```
