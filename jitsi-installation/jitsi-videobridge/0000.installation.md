# jitsi-videobridge

## 查看命令

```
dpkg -L jitsi-videobridge2
find / -name "*videobridge*"
find / -name "*jvb*" | grep -v "prosody"

ll /var/lib/dpkg/info/jitsi-videobridge2.*
sz /var/lib/dpkg/info/jitsi-videobridge2.list
```
